# Aurora Health Companion App

Aurora is an AI-powered mobile health companion built with React Native and Expo. It helps users track hydration, sleep, mood, habits, nutrition, and wellness progress with secure authentication and MongoDB-backed data persistence.

## Features

- User registration and login
- Persistent login session
- Health profile with BMI calculation
- Hydration tracking
- Sleep tracking
- Mood tracking
- Habit tracking
- Nutrition and macro tracking
- Health dashboard and analytics
- Local storage using AsyncStorage
- Backend integration using REST APIs
- MongoDB auto-save and data loading
- Logout and reset data flow

## Tech Stack

- React Native
- Expo
- JavaScript
- AsyncStorage
- Axios
- React Native Chart Kit
- Expo Notifications
- Node.js
- Express.js
- MongoDB
- Mongoose
- Render
